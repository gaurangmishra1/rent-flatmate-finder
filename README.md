# Rent & Flatmate Finder

## Overview
A full-stack web application that helps tenants find rental properties and flatmates while allowing owners to manage listings and interests.

## Tech Stack
- React
- TypeScript
- Node.js
- Express.js
- Prisma ORM
- PostgreSQL
- Socket.IO
- JWT Authentication

## Features

### Authentication
- User Registration
- User Login
- JWT Authentication
- Role-Based Access (Owner/Tenant)

### Owner
- Create Listings
- View Interested Tenants
- Accept/Decline Interests

### Tenant
- Create Tenant Profile
- Browse Listings
- Express Interest
- View My Interests

### Other Features
- Compatibility Score
- Real-Time Chat using Socket.IO
- PostgreSQL Database
- Prisma ORM

## Installation

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
